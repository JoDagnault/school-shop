# School Shop Frontend

React, TypeScript, and Vite frontend for School Shop.

## Scripts

- `pnpm dev` starts the local Vite dev server.
- `pnpm build` type-checks and creates a production build.
- `pnpm lint` runs ESLint.
- `pnpm preview` serves the production build locally.

## Development

```sh
pnpm install
pnpm dev
```
